## 贡献指南

### 文档命名规范

| 类型 | 命名格式 | 示例 |

| 用户指南 | 中文命名 | 用户指南.md |

### 文档更新流程

1. 新建文档时选择正确的目录
2. 在文档开头添加最后更新日期
3. 更新本索引文档的导航表格
4. 完成的临时文档及时归档到 `archive/` 目录

## 常见问题

### 如何快速上手？

1. 阅读 [README](../README.md) 了解项目概况
2. 参考 [用户指南](guides/用户指南.md) 进行配置
3. 遇到问题查看故障排除章节

### 如何参与开发？

1. 阅读 [开发工作流](reference/WORKFLOW.md) 了解开发流程
2. 按照验收标准提交代码
3. 确保所有测试通过
